足球俱乐部管理系统囊括俱乐部（包含名称和总资金）、教练（包含员工编号、姓名、年薪、能力值）、球员（包含前锋、中场、后卫、守门员[其中各自有球衣号码、姓名、年薪、能力值、个人特长]）等与俱乐部有关人员的数据集。要求设计一个程序，实现对足球俱乐部的管理。具体管理功能如下：
